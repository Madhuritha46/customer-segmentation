## 🧠 Overview
This project is a **Machine Learning web application** that performs **Customer Segmentation** using the **K-Means Clustering algorithm**. It helps group customers based on behavior such as income and recency, enabling businesses to design better marketing strategies.

The app is built using **Streamlit** and deployed on **Streamlit Cloud**.

---

## 🎯 Objective
To segment customers into meaningful groups so that businesses can:
- Understand customer behavior
- Improve targeted marketing strategies
- Increase sales and customer engagement

---

## 📂 Dataset
- **Name:** marketing_campaign.csv  
- **Type:** Marketing / Customer Behavior Dataset  
- **Format:** Tab-separated file (`sep="\t"`)  
- **Key Features Used:**
  - Income
  - Recency  

---

## ⚙️ Tech Stack
- Python 🐍  
- Streamlit 📊  
- Pandas  
- Matplotlib  
- Scikit-learn  

---

## 🤖 Machine Learning Model
- **Algorithm:** K-Means Clustering  
- **Scaling:** StandardScaler  
- **Purpose:** Group customers into clusters based on similarity  

---

## 📊 Features of the App

### 📌 Dashboard
- Interactive visualization of customer clusters
- Color-coded segmentation plot

### 📁 Data View
- Displays clustered dataset
- Shows real-time processed data

### 🔮 Prediction Tool
- Input customer details (Income, Recency)
- Predict customer segment instantly

---

## 🏗️ Project Workflow
1. Load dataset  
2. Clean missing values  
3. Select important features  
4. Scale data using StandardScaler  
5. Apply K-Means clustering  
6. Visualize clusters  
7. Build Streamlit web application  
8. Deploy on Streamlit Cloud
